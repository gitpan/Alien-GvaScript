Ajax Autocompleter

This autocompleter example is bound to a complete list of countries served by the perl HTTP service at http://localhost:8081/ac/country/ that responds by an array: [{value: ..., label: ..., code: ...}, ... ]

Autocomplete 1 : Normal :

Autocomplete 2 : (typeAhead: false)

Autocomplete 3 : (strict: true, blankOK: true)

Autocomplete 4 : (strict: true, blankOK: false)

Autocomplete 5 : (strict: true, blankOK: true, autoSuggest: false, typeAhead: true)

Autocomplete 6 : (strict: true, blankOK: true, autoSuggest: false, typeAhead: false)
